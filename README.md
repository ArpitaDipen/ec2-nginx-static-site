# EC2 + NGINX Static Site Deployment

## Project Overview
This project demonstrates hosting a **static website on an AWS EC2 instance** using **NGINX** as the web server. The site is accessible via the EC2 public IP (or custom domain if configured) and showcases basic HTML content.

## Features
- Hosted on **Ubuntu 22.04 LTS EC2 instance**
- **NGINX** used as the web server
- Secure server setup (ports 22 for SSH, 80 for HTTP)
- Optional: HTTPS setup with **Let’s Encrypt**
- Static HTML content (`index.html`) deployed

## Technologies Used
- **AWS EC2**
- **NGINX**
- **HTML/CSS**
- **Linux CLI**
- (Optional) **Let’s Encrypt SSL**
